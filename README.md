Welcome to cafetch, a CLI system informer written in the C programming language.

Using GNU GPLv3 License.

(note this doesnt list system hardware components like CPU, GPU, and memory.)

## Dependencies:
```
go
git (for install)
```


## Install guide

- Clone the repository: ```git clone https://github.com/catowner-sudo/cafetch```
- Navigate to the directory: ```cd cafetch```
- Build and install: ```makepkg -si```
- Now you can run ```cafetch``` anywhere!


## Features

- Displays basic system information, including:
  - Desktop Environnment
  - Packages
  - Shell
  - Kernel
  - And more!
